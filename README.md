# VDMX VJ

Homebrew Tap for VJ related mac softwares.

## How to use

`brew install sisoe24/vj/vdmx`

Or `brew tap sisoe24/vj` and then `brew install vdmx`.

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
